Execution Guide
To ensure the project runs correctly in the Visual Studio environment, please follow these steps:

Set Platform: Set the Solution Platform to x86 in the top toolbar.

Set Startup Project: Right-click the project you wish to execute (e.g., q3) in the Solution Explorer and select Set as Startup Project.

Run: Press F5 to build and run the program.

Note: This project is configured specifically for x86. Running it in x64 or without setting the correct startup project may result in build errors or execution failures.
